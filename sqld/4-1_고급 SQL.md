# ⛰️ 4단계: 마의 구간, 고급 SQL (윈도우 함수와 그룹화)

2단계에서 배운 기본 함수들이 '개인전'이고, 3단계의 `GROUP BY`가 '팀전'이라면, 4단계의 고급 함수들은 **'팀전의 결과를 개인에게 달아주는 마법'**과 **'엑셀의 피벗 테이블 기능'**이라고 생각하시면 이해가 쉽습니다.

---

## 1. 🪟 윈도우 함수 (Window Function): "내 순위는 몇 등일까?"

기존의 `GROUP BY`는 데이터를 묶어버리기 때문에 개별 데이터의 상세 정보가 화면에서 사라집니다. 하지만 윈도우 함수를 쓰면 **원래 데이터는 그대로 살려두면서, 옆에 새로운 칸을 하나 파서 '그룹 내 순위'나 '합계'를 몰래 적어줄 수 있습니다.**

### 🏆 3대 순위 함수 (시험 출제율 100%)
급여가 각각 [300, 200, 200, 100]인 4명의 직원이 있다고 가정해 봅시다.

1. **`RANK()`:** 공동 순위가 있으면 다음 순위를 건너뜁니다. (가장 일반적인 순위)
   👉 결과: 1등, **2등, 2등, 4등** (3등이 없음)
2. **`DENSE_RANK()`:** 공동 순위가 있어도 다음 순위를 촘촘하게(Dense) 이어서 매깁니다.
   👉 결과: 1등, **2등, 2등, 3등**
3. **`ROW_NUMBER()`:** 공동 순위를 인정하지 않고 무조건 고유한 번호를 부여합니다.
   👉 결과: 1등, **2등, 3등, 4등** (누가 2등이 될지는 시스템 마음이나 추가 정렬 기준에 따름)

* **핵심 문법:** `순위함수() OVER (PARTITION BY 그룹기준 ORDER BY 정렬기준)`
  * 예: `RANK() OVER (PARTITION BY 부서 ORDER BY 급여 DESC)` -> "부서별로 팀을 나눈 뒤, 급여가 높은 순서대로 랭킹을 매겨라!"

## 2. 🧮 그룹 함수 (Group Function): "엑셀 소계/합계 자동 계산"

`GROUP BY`의 업그레이드 버전입니다. 단순히 그룹화만 하는 게 아니라, 우리가 엑셀에서 자주 쓰는 **'소계'와 '총합계'를 한 줄로 뽑아주는 마법의 함수들**입니다.

* **`ROLLUP (A, B)`:** A별 소계를 구하고, 마지막에 전체 총합계를 구합니다. 
  * **특징:** 방향성이 있어서 `ROLLUP(A, B)`와 `ROLLUP(B, A)`는 결과가 다릅니다! (대분류 -> 중분류 순서로 묶음)
* **`CUBE (A, B)`:** 결합 가능한 **모든 경우의 수**에 대해 다차원 소계와 총합계를 구합니다.
  * **특징:** A별 소계, B별 소계, 전체 총합계까지 무식하게 다 구합니다. (데이터가 엄청나게 뻥튀기됨)
* **`GROUPING SETS (A, B)`:** 내가 원하는 특정 그룹의 소계만 쏙쏙 골라서 구합니다. 총합계는 알아서 구해주지 않습니다.

## 3. 🌳 계층형 질의: "조직도와 족보 그리기"

데이터베이스 안에 있는 '사원'과 '매니저', 혹은 '게시글'과 '댓글'처럼 **꼬리에 꼬리를 무는 부모-자식 관계(트리 구조)**를 찾아가는 쿼리입니다.

* **`START WITH`:** 트리의 뿌리(시작점)를 지정합니다. (예: `START WITH 매니저 IS NULL` -> 사장님부터 시작!)
* **`CONNECT BY`:** 부모와 자식이 어떻게 연결되는지 규칙을 정합니다.
* **`PRIOR` (매우 중요 🌟):** '이전의, 부모의'라는 뜻입니다.
  * `CONNECT BY PRIOR 사원번호 = 매니저번호` 
    👉 "방금 읽은 부모의 사원번호가, 다음에 읽을 자식의 매니저번호인 사람을 찾아라!" (사장님 -> 말단 사원으로 내려가는 **순방향 전개**)

## 4. 🚨 초보자 단골 함정 (트러블슈팅 가이드)

### ❌ 함정 1: 순위 함수의 결과값 비교
* **출제 유형:** 표를 주어주고 쿼리를 실행했을 때 3번째 줄에 들어갈 숫자는 무엇인가?
* **해결책:** 눈으로 대충 풀면 무조건 틀립니다. 시험지에 `RANK`, `DENSE_RANK`, `ROW_NUMBER` 3가지를 직접 옆에 적어보세요. 특히 **공동 2등 다음이 3등인지 4등인지**를 묻는 문제가 매번 나옵니다.

### ❌ 함정 2: `ROLLUP`의 순서 함정
* **출제 유형:** 다음 중 결과표와 동일한 결과를 내는 쿼리는?
* **정답 및 해설:** `ROLLUP(부서, 직급)`을 하면 '부서별 소계'와 '전체 총계'는 나오지만 '직급별 소계(부서 무시)'는 나오지 않습니다. 반대로 `ROLLUP(직급, 부서)`를 하면 '직급별 소계'가 나옵니다. 표에 어떤 소계가 남아있는지 보고 ROLLUP의 순서를 맞춰야 합니다.
